# 📊 Pink Morsel Sales Dashboard

An interactive data visualization dashboard built using Dash and Plotly to analyze the impact of a price increase on Pink Morsel sales.

## 🚀 Features

* 📈 Line chart showing sales over time
* 🔘 Region filter (North, South, East, West, All)
* 📊 Data processed using Pandas
* 🎨 Styled UI with CSS
* ✅ Automated testing using Pytest and Dash Testing

## 🧠 Problem Statement

Analyze whether sales increased or decreased after the price hike on **January 15, 2021**.

## 🛠 Tech Stack

* Python
* Pandas
* Dash
* Plotly
* Pytest
* Selenium

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
Open in browser:
http://127.0.0.1:8050/
```

## 🧪 Run Tests

```bash
pytest
```

## 📌 Result

The visualization clearly shows that **sales increased after the price change**, indicating strong product demand.
